# Frontend - Disk Scheduling Algorithm Simulator

React frontend for the Disk Scheduling Algorithm Simulator.

## Setup

```bash
npm install
```

## Running the Development Server

```bash
npm start
```

The application will open at `http://localhost:3000`

## Building for Production

```bash
npm run build
```

## Project Structure

- `src/App.js`: Main application component
- `src/components/`: React components
  - `InputPanel.js`: User input form
  - `VisualizationPanel.js`: Chart visualizations
  - `ResultsPanel.js`: Simulation results display
  - `ComparisonPanel.js`: Algorithm comparison view
  - `Header.js`: Application header

## Technologies

- React 18
- Recharts for visualizations
- Axios for API calls
- CSS3 for styling
